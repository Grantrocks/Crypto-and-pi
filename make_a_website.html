Meta Tags

The second most important part of making a website the META TAGS The meta tags are the tags that tell you web browser what to display and a little bit more information such as the lanuage. The meta tags that you need to add are the language meta tags, charset meta tag, author meta tag, keyword meta tag, decritption meta tag, icon meta tag, title meta tag, and the viewport meta tag.

These are the meta tags and only a couple are very important such as the title description and keyword meta tag.

Text Tags

The third thing you need for your website is content. And in order to do that you need some text tags such as H1, H2, H3, P, OL, LI The h1 text tag makes the text within it really big and usually this is what you would use for the main title of the page, but dont mix this up with the title meta tag. The second text tag that you should have is the h2 tag. This tag makes the content withing it big but not as big as the h1 tag. The next tag you should use is the h3 tag. This text tag makes the content look like a heading.
